# CardGame
**简介**

  本文件是一个翻牌游戏，整个文件夹下载下来直接运行main.py就可以

**使用技术**  
全部采用Python语言编写，GUI用tkinter，数据库用的sqlite3来管理单词,

**支持功能：**  
1.允许查看排行榜，可以看到其他用户的最高积分和历史总积分  
2.允许自己选择难度  
3.可以添加单词，只需要在words.txt里面放入想导入的单词，然后点击import_words.py运行脚本即可导入   
**彩蛋：** 
每次开始游戏的时候，会随机为你分配一张卡牌图片  
**后续可能不定期更新：**  
1.现在只能添加单词，无法替换单词，后续有时间会更新用words.txt文件替换原有单词库的功能
2.游戏功能后续可能会添加
